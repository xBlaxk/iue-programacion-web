Protocolos

HTTP

Es el método principal de intercambio de información en la web. Un cliente (generalmente un navegador web) envía una solicitud a un servidor, que luego devuelve una respuesta. Esta respuesta generalmente es una página web, una imagen, etc.

TCP/IP

Este es el conjunto fundamental de protocolos que forman la base de Internet. TCP se ocupa de la formación y el control de conexiones entre computadoras, mientras que IP se ocupa de la ruta que toman los datos a través de la red.

DNS

Es la "guía telefónica" de Internet, convierte los nombres de dominio amigables para los humanos (como "www.google.com") en direcciones IP que las computadoras pueden usar para comunicarse entre sí.

CDN

Es una red de servidores que entrega contenido web a un usuario basándose en su ubicación geográfica, el origen del sitio web y un servidor de entrega de contenido. Ayuda a aumentar la velocidad de entrega del contenido de los sitios web a los usuarios.

WWW

Es el sistema de documentos de hipertexto interconectados que se acceden a través de Internet. Un documento de la web, que generalmente se denomina página web, es generalmente un archivo de texto enriquecido con hipervínculos a otros documentos de la web.

FTP

Es un protocolo estándar de Internet que transfiere archivos de un host a otro a través de una red basada en TCP, como Internet.

HTML

Es el lenguaje de marcado estándar para la creación de páginas web. Define la estructura de una página web y consiste en una serie de elementos que indican a los navegadores cómo mostrar el contenido.
